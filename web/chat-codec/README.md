# CodecShare Chat

**P2P encrypted chat** using [codec-share](https://github.com/lcmonteiro/codec-share) fountain codes + WebRTC mesh.

```
Message -> codec-share encode -> N coded frames -> WebRTC parallel channels -> codec-share decode -> Original
```

- **No server stores messages** -- pure P2P via WebRTC mesh
- **Fountain codes** -- no retransmissions, no ordering needed
- **Parallel DataChannels** -- 4 channels per peer, frames distributed round-robin
- **Room token = encryption key** -- token seeds the fountain code; only peers with the same token decode
- **Submodule** -- codec-share lives under `wasm/codec-share/` as a git submodule

## Architecture

| Layer | Technology |
|---|---|
| Transport | WebRTC via PeerJS (mesh) |
| Encoding | codec-share -> WASM (Emscripten) |
| Discovery | Deterministic hub peer ID from token hash |
| Fallback | JS identity codec when WASM unavailable |
| Deploy | GitHub Pages (static files) |

## CLI version

Chat directly from the terminal -- same P2P mesh, same fountain-code encryption.

```bash
cd web/chat-codec/cli
npm install

# Join a room
node cli-chat.js --token myroom

# With nickname
node cli-chat.js --token myroom --nick M

# Help
node cli-chat.js --help
```

**Keys:** Enter to send, Esc/Ctrl+C to quit, PgUp/PgDn to scroll.

The CLI uses a pure-JS codec and is compatible with the browser version -- same token = same room.

## Quick start (browser)

```bash
# 1. Clone with submodule
git clone --recurse-submodules git@github.com:lcmonteiro/mcking-codespace.git
# or if already cloned: git submodule update --init web/chat-codec/wasm/codec-share

# 2. Install Emscripten
git clone https://github.com/emscripten-core/emsdk.git
cd emsdk && ./emsdk install latest && ./emsdk activate latest
source ./emsdk_env.sh

# 3. Build WASM
cd mcking-codespace/web/chat-codec && bash wasm/build.sh

# 4. Dev server
npx serve public/ -l 3000
```

> Without WASM, the app uses a JS identity codec -- UI and WebRTC still work.

## Usage

1. Open the app
2. Enter a **Room Token** (any phrase, or click the dice for a random one)
3. Optional: set a nickname
4. **Join Room** -- first joiner creates the room (hub)
5. Share the token with others -- they join the same room
6. Chat! Messages are fountain-coded and sent over 4 parallel channels
